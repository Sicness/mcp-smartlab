# mcp-smartlab

MCP-сервер для получения финансовых данных с smart-lab.ru (облигации, акции, дивиденды).

## Инструменты

### Облигации

- **`search_bonds`** — основной скринер облигаций с фильтрами (рейтинг, дюрация, сектор, флоатер, амортизация, бессрочные). Поддерживает типы: corporate, ofz, subfederal, currency, other
- **`get_top_yield_bonds`** — быстрый поиск самых доходных облигаций инвестиционного грейда (дефолт: BBB+, дюрация до 5 лет)
- **`get_bond_chart_data`** — облегчённые данные yield/duration/rating без пагинации (из JS-переменной aBondsChartData)
- **`get_bond_details`** — детали одной облигации (купонный график)
- **`compare_bonds`** — сравнение нескольких облигаций по secid
- **`get_bond_sectors`** — список доступных секторов для фильтрации

### Акции

- **`search_shares`** — скринер акций MOEX (цена, объём, изменения за день/неделю/месяц/YTD/12м, капитализация)
- **`get_shares_fundamental`** — фундаментальные данные (P/E, P/S, P/B, EV/EBITDA, дивдоходность, рентабельность EBITDA, долг/EBITDA)

### Дивиденды

- **`get_upcoming_dividends`** — календарь ближайших дивидендов (тикер, сумма, доходность, даты отсечки и выплаты)
- **`get_dividend_history`** — история дивидендных выплат по конкретному тикеру

## Особенности

- Данные парсятся из HTML-таблиц smart-lab.ru (серверный рендеринг, без API)
- In-memory кэш: 1 час для скринера, 4 часа для фундаментала и дивидендов
- Rate limiting: макс. 3 одновременных запроса, задержка 0.5с между запросами
- Таблица облигаций адаптируется к разным типам (corporate/OFZ/subfed имеют разные колонки)

## Фильтры облигаций

| Параметр | Значения |
|---|---|
| `rating` | AAA, AA+, AA, AA-, A+, A, A-, BBB+, BBB, BBB-, BB+, BB, BB-, B+, B, B- |
| `sector` | bank, oil, finance, leasing, build, it, metal, chemistry, food, retail, transport, telecom, energy |
| `floater` | "" (все), "yes" (только флоатеры), "no" (только фиксированные) |
| `amortization` | "" (все), "yes" (с амортизацией), "no" (без амортизации) |
| `perpetual` | "" (все), "yes" (бессрочные), "no" (с погашением) |
| `duration_from` / `duration_to` | число (годы) |
| `coupon_frequency` | 1, 2, 4, 12 (раз в год) |
